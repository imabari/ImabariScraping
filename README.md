# ImabariScraping
今治市の情報をスクレイピング

## imabari119_bot.py

えひめ医療情報ネットの今治市地区の当番医案内から今日の医療機関をツイート

```sh
npm install phantomjs
pip3 install selenium
pip3 install beautifulsoup4
pip3 install python-twitter
```

## refuge_rss.py

避難情報をRSS配信

```sh
pip3 install pytz
pip3 install beautifulsoup4
pip3 install feedgenerator
pip3 install bottle
```

### 避難情報（避難準備情報、避難勧告、避難指示情報）
/urge/

### 避難所情報
/shelter/

## tamagawadam_bot.py
